<a name="introduction">🤖 Introduction</a>

Welcome to ImageX! An AI-powered mobile app that generates creative and high-quality images from user inputs.
Built with React Native for the frontend, Node.js + Express for the backend, and powered by the Replicate API, ImageX provides a smooth, responsive, and scalable solution for AI-based image generation.


<a name="tech-stack">⚙️ Tech Stack</a>

React Native – Frontend mobile app

Tailwind CSS – Styling and responsive UI

Node.js – Backend server

Express – REST API routes

Replicate API – AI image generation engine


<a name="features"> 🔋 Features</a>

AI Image Generation – Create unique images from text prompts.

Responsive Mobile UI – Works on Android and iOS devices.

Aspect Ratio Selection – Customize image dimensions.

Preview & Download – View generated images and save them locally.

Re-generate Images – Easily try multiple variations of your prompts.

Lightweight & User-friendly – Clean interface for seamless user experience.
